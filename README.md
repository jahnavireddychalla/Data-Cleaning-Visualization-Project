# Data Cleaning and Visualization Project

## Objective

The objective of this project is to analyze and visualize student performance data using Python. The project focuses on understanding the dataset, checking data quality, generating statistical summaries, and creating meaningful visualizations.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Visual Studio Code (VS Code)
* GitHub

## Dataset

Students Performance Dataset

## Tasks Performed

1. Loaded the dataset using Pandas.
2. Displayed the first few rows of the dataset.
3. Explored dataset structure and information.
4. Checked the number of rows and columns.
5. Identified column names.
6. Checked for missing values.
7. Checked for duplicate records.
8. Generated statistical summaries.
9. Created visualizations to analyze student performance.

## Visualizations Created

### 1. Math Score Distribution

Shows the distribution of students' mathematics scores.

### 2. Gender Distribution

Shows the number of male and female students in the dataset.

### 3. Math Score by Gender

Compares mathematics performance across genders using a box plot.

## Key Findings

* No missing values were found in the dataset.
* No duplicate records were found.
* Student scores are distributed across a wide range.
* The dataset contains both male and female students in significant numbers.
* Mathematics performance can be analyzed and compared across genders.
