# Data Cleaning and Visualization Project

## Objective

The objective of this project is to analyze and visualize student performance data using Python.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* VS Code

## Dataset

Students Performance Dataset

## Tasks Performed

1. Loaded the dataset.
2. Explored dataset structure.
3. Checked for missing values.
4. Checked for duplicate records.
5. Generated statistical summary.
6. Created visualizations.

## Visualizations

* Math Score Distribution
* Gender Distribution
* Math Score by Gender

## Conclusion

The dataset was successfully analyzed and visualized. The project demonstrates basic data cleaning and visualization techniques using Python.
